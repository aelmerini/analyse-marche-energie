# Analyse du marché de l’énergie

## Contexte
Projet réalisé dans le cadre d’un stage en analyse marketing chez ENGIE Local Energy Infrastructures.

## Objectifs
- Comprendre les tendances du marché de l’énergie
- Identifier les opportunités et risques
- Appuyer la prise de décision marketing

## Méthodologie
- Analyse de données marché
- Études concurrentielles
- Synthèse des insights clés

## Livrables
- Analyse chiffrée
- Graphiques et dashboards
- Recommandations marketing
