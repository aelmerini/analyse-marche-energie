# Analyse du marché de l’énergie

## Contexte
Projet d’analyse du marché de l’énergie réalisé dans le cadre d’un stage en analyse marketing.

Les données utilisées dans le cadre professionnel sont confidentielles et ne sont pas partagées dans ce repository.

## Objectifs
- Comprendre les grandes tendances du marché de l’énergie
- Identifier les principaux enjeux sectoriels
- Structurer une démarche d’analyse marketing

## Méthodologie
- Étude de sources publiques et sectorielles
- Analyse de données agrégées
- Recherche documentaire
- Synthèse des insights clés

## Contenu du projet
Ce repository présente :
- la structure du projet d’analyse
- la démarche méthodologique
- des exemples ou modèles de travail (non sensibles)

Aucune donnée interne ou confidentielle n’est incluse.

## Confidentialité
Ce projet respecte les règles de confidentialité de l’entreprise.
Les livrables internes et les données réelles restent sur les outils professionnels dédiés.
